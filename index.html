Conditionals

Notes

Conditionals are expressions that evaluate to either true or false or simply statements which run under certain conditions.
AnÂ operatorÂ in aÂ programmingÂ language is a symbol that is used to perform operation on data. It acts as connector and indicate type of operation being carried out.
Operator are broadly classified into 3 types: Unary, Binary, Ternary.
If statement: The simplest type of conditional statement. It is used to decide whether a certain statement or block of statements will be executed or not i.e if a certain condition is true then a block of statement is executed otherwise not. 

Syntax
  if(condition) 
  {
      // Statements to execute if
      // condition is true
  }

Else statement: We can use theÂ elseÂ statement withÂ ifÂ statement to execute a block of code when the condition is false. 

Syntax
  if (condition)
  {
      // Executes this block if
      // condition is true
  }
  else
  {
      // Executes this block if
      // condition is false
  }

Nested if : A nested if is an if statement that is the target of another if statement. Nested if statements means an if statement inside another if statement. 

Syntax
  if (condition1) 
  {
      // Executes when condition1 is true
      if (condition2) 
      {
        // Executes when condition2 is true
      }
  }

Else if statement: This is used when we want to deal with more than 2 outcomes of a condition. The execution takes place in top-down fashion i.e if 1st condition is false it moves down to the next until a true condition is found after which it stops checking and executes the statement of that condition. It is also known as if-else if ladder. 

Syntax
  if (condition)
      statement;
  else if (condition)
      statement;
  else
      statement;

The break statement: Used to terminate a flow of operations if a certain condition is met.(i.e to exit from a loop) 

Syntax
  break;

The continue statement: Used to skip the current operation and proceed to next if a certain condition is met. 

Syntax
  continue;

The goto statement: Also referred to as unconditional jump statement can be used to jump from one point to another within a function. 

Syntax
  goto label;

The return statement: Returns the flow of the execution to the function from where it is called.Â  

Syntax
  return[expression];

The Switch-Case: Switch case statements are a substitute for long if statements that compare a variable to several integral values. It provides an easy way to dispatch execution to different parts of code based on the value of the expression. 

Syntax
  switch (n)
  {
      case 1: // code to be executed if n = 1;
          break;
      case 2: // code to be executed if n = 2;
          break;
      default: // code to be executed if n doesn't match any cases
  }

Looping

Notes

A loop statement allows us to execute a statement or group of statements multiple times.
An entry control loop checks the condition at the time of entry and if the condition or expression (statement that has value) becomes true then control transfers into the body of the loop. Example: for loop and while loop
An exit control loop checks the condition for exit and if given condition for exit evaluate to true, control will exit from the loop body or else control will enter again into the loop. Example: Do while loop
A while loop statement repeatedly executes a target statement as long as a given condition is true. 

Syntax
  initialization expression;
  while (test_expression)
  {
      // statements
    
    update_expression;
  }

A for loop is a repetition control structure that allows you to efficiently write a loop that needs to execute a specific number of times. 

Syntax
  for (initialization expr; test expr; update expr)
  {    
    // body of the loop
    // statements we want to execute
  }

A do-whileÂ loop is similar to a while loop, except the fact that it is guaranteed to execute at least one time. It checks its condition at the bottom of the loop. 

Syntax
  initialization expression;
  do
  {
  // statements

  update_expression;
  } while (test_expression);

A loop can be used inside of another loop. This is known as nesting. Nested for loops are most commonly used in problem solving. 

Syntax for Nested For loop:
  for ( initialization; condition; increment ) {

    for ( initialization; condition; increment ) {
        
        // statement of inside loop
    }

    // statement of outer loop
  }

Syntax for Nested while loop:
  while(condition) {

    while(condition) {
      
      // statement of inside loop
    }

    // statement of outer loop
  }

Syntax for Nested do-while loop:
  do{

    do{
      
      // statement of inside loop
    }while(condition);

    // statement of outer loop
  }while(condition);

Arrays

Notes

An array is a collection of similar data items stored at contiguous memory locations and elements can be accessed randomly using indices of an array. 

Syntax to declare an array :
  dataType arrayName[arraySize];
Syntax to initialize an array :
  dataType arrayName[arraySize] = {d1, d2, d3...., dn};

Array elements are accessed by using an integer index. Array index starts with 0 and goes till size of array -1.
One-dimensional array is a group of elements having the same datatype and are stored in a linear form.
Multidimensional arrays in simple words as array of arrays.
Two-dimensional arrays can be referred to as arr[i][j] wherein i is considered to be the row number and j is considered to be column number. In this data is stored in rows and columns. 

Syntax of 1-D array:
  data_type array_name[size];
Syntax of 2-D array :
  data_type array_name[row][columns];
Syntax of 3-D array:
  data_type array_name[size1][size2][size3];

Sorting means arranging the data in a particular fashion, which can be increasing or decreasing.
Searching is the process of finding the required data in the array.
There are two techniques of searching elements in array:
Sequential Search : We compare targeted element with each element of the array. If the element is found then its position is displayed.
Binary Search : Finding the required element in a sorted array by repeatedly halving the array and searching in the half.
Vector are same as dynamic arrays with the ability to resize itself automatically when an element is inserted or deleted, with their storage being handled automatically by the container.
Vector elements are placed in contiguous storage so that they can be accessed and traversed using iterators. In vectors, data is inserted at the end. 

Syntax to declare an vector :
    vector<data_type> vector_name(size);

Strings

Notes

A string variable contains a collection of characters surrounded by double quotes. 

Syntax
     string string_name = "data";
Example
    string x = "Hello World";

String Concatenation : The + operator can be used between strings to add them together to make a new string. Or you can also concatenate strings with the append() function. 

Example
    string x = "Hello ";
    string y = "World";
    string a = x + y;   // Hello World
    string b = x.append(y); // Hello World

String length : To get the length of a string, use the length() or size() function 

Syntax
    string_name.length();
    string_name.size();
Example
    string x = "Hello World";
    cout << x.length();   // 11
    cout << x.size();   // 11

Access String : You can access the characters in a string by referring to its index number inside square brackets [].
Note: String indexes start with 0: [0] is the first character. [1] is the second character, etc.
User Input Strings : The getline() function to read a line of text. It takes cin as the first parameter, and the string variable as second. 

Syntax
    getline (cin, string_name);
